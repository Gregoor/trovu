trovu.net
Web shortcuts, social, private.
Namespaces:
Language: Country:

Example queries

g berlin
search Google for "berlin"
gd london, liverpool
find a route on Google Maps from London to Liverpool

Settings via URL parameters

Example:

#language=en
&country=.de
&namespaces=o,en,.de,jorges
&namespace[jorges]=https%3A%2F%2Fraw.githubusercontent.com%2Fgeorgjaehnig%2Ftrovu-user%2Fmaster%2Fshortcuts%2F{%25keyword}%2F{%25argumentCount}.txt
language
2-letter language code. Will be used to replace {$language} within the shortcut URLs.
country
2-letter country code. Will be used to replace {$country} within the shortcut URLs.
namespaces
List of namespaces to use, separated by comma.
namespace[someNamespace]
URL template for fetching shortcuts for someNamespace. someNamespace must be listed in namespaces.

Add to browser

/process/?#query=%s
