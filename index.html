Using Advanced settings via Github. Basic settings disabled.

Set advanced settings via your Github account:

  1. Fork trovu-data-user.
  2. Adjust config.yml to your needs.
  3. Set URL parameter trovu.net/#github=YOUR_GITHUB_USERNAME and reload.
  4. Check Debug tab if needed.

Not matching? Github may need some minutes to update your raw config.yml.

Example queries

gd london, liverpool
find a route on Google Maps from London to Liverpool
gfl txl, lpa
Google Flights search from Berlin to Las Palmas
w berlin
open Wikipedia article for Berlin (in language from settings)
fr.w berlin
override language from settings
g berlin
search Google for "berlin"

About

  • privacy: all queries are processed in the client, not server
  • social: benefit from curated shortcuts in the general repo, organized by namespaces.
  • freedom: setup your own personal shortcuts in your private repo
