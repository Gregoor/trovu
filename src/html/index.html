Using Advanced settings via Github. Basic settings disabled.

Set advanced settings via your Github account:

  1. Fork trovu-data-user.
  2. Adjust config.yml to your needs.
  3. Set URL parameter trovu.net/#github=YOUR_GITHUB_USERNAME and reload.
  4. Check Debug tab if needed.

Not matching? Github may need some minutes to update your raw config.yml.

You may also set Debug mode with trovu.net/#debug=1.
